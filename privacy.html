Our Privacy Policy



1. What data do we collect?

Our bot have a feature that can change the bots prefix. When user change the bots user, We collect their username, id, guild name/id and the prefix.


2. Why do we need our data?

We collect your data becasue we need to memorize your desired prefix for the bot.


3. How do we use the data?

When the user execute a command there's a desired prefix for the command and we use user data to find that data. We use username, user id and guild id


4. What are the 3rd party services that we're using?

We use mongodb company to hold your data. It's a trusted company by millions of users.


5. How can user have that data removed?

We store our data in a databse so if user want to remove his/her data the user can join to our official discord server and make a ticker requesting to remove their data from the database. Server link - https://discord.gg/muUcvsV
